## What will I learn?

This module focuses on

1. introduction to the CPU scheduling and description of several CPU-scheduling algorithms.
2. evaluation of selected CPU-scheduling algorithm for a particular system based a defined criterion.
3. description of deadlocks, which prevent sets of concurrent processes from completing their tasks.
4. Introduction to several different methods for preventing or avoiding deadlocks in a computer system.

This forms the core component to the subject.

## What will I be able to do at the end of this module?

At the end of this module, you should be able to answer these questions:

1. What are the scheduling criteria?
2. Can you describe those scheduling algorithms including FCFS, SIF, SRTF, and RR?
3. What is the multilevel feedback queue?
4. In resource-allocation graph, can you give two examples, one for graph with a deadlock and the other for graph with a cycle but no deadlock?
5. Can you use your own words to describe the Banker’s algorithm?
6. Can you use your own words to describe the deadlock detection algorithm?
