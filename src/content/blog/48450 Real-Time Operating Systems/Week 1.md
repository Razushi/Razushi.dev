## What will I learn?

This module focuses on

1. basic concept of computer systems and major structure of operating systems
2. description on the services that an operating system provides users, processes, and other systems
3. Systems level of processes for essential OS programming and operations

This forms the introduction to the subject.

## What will I be able to do at the end of this module?

At the end of this module, you should be able to answer these questions:

1. What are the operating system services?
2. What the standard C library, API system call on OS.
3. What is the folk, wait, kill and exit processes?
4. What does POSIX mean?
## Key vocabulary in this module:

word: OS structure, OS service, OS programming and basic concept

In this module, you have learnt about basic concepts of operating system services, standard C library for API system call on the folk, wait, kill and exit processes.
