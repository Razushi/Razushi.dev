## What will I learn?

This module focuses on

1. basic concept of process state, CPU switch, and threads.
2. description of various features of processes, including scheduling, creation, and termination.
3. Interposes communication using shared memory and message passing.

This forms the core component to the subject.

## What will I be able to do at the end of this module?

At the end of this module, you should be able to answer these questions:

1. What are the concepts of Process state, CPU switch, and Thread?
2. How the fork system call works?
3. How to describe the process scheduling, process creation and termination.
4. What are the main concepts of IPC including the message synchronization and shared memory?
5. What are the key points of Real-time Operating Systems for Embedded System?
## Key vocabulary in this module:

Key words: Process state, CPU switch, Thread, Scheduling, Process creation, IPC and RTOS functions.
