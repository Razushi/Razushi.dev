### Focus Questions  
As you read CCNA ITN Module 11.4 - 11.10 check you can understand and complete the  
Following:  
1. Understand why subnetting is important  
2. Know that every IPv4 Address has a network portion and a host portion. Subnetting is taking the high order bits from the host portion and making them network bits. This reduces the size (number of host addresses) of these networks which are called subnets.  
3. How to calculate subnets.  
4. Determining the network and subnetwork address in decimal and binary.  
5. Calculating the number of valid host addresses per subnet.  
6. Understanding and calculating subnet masks.  
7. Variable Length Subnet Mask (VLSM) subnets – subnetting a subnet.  
8. Planning Addressing Schemes
