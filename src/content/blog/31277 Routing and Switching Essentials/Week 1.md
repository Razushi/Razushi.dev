### Focus Questions  
1. Understand place value in both decimal and binary number systems.  
2. Convert binary to decimal and decimal to binary in an 8 bit environment.  
3. Understand the concept of a subnet mask. Be able to know that all IP Addresses have a network portion and a host portion.  
4. A LAN or subnetwork is a collection of devices with their gateway as a router interface.  
5. Understand the difference in structure between a unicast, broadcast and multicast IPv4 Address.  
6. Identify Private Addresses. What is the main feature of these addresses?  
7. Identify Class Blocks of IPv4 Addresses – Class A, B, C, D or E.
